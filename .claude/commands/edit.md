---
description: "Revision, scrubbing, and cutting for written content. Use when asked to edit, revise, tighten, cut, polish, clean up, or when running a revision pass on a draft."
---
This skill assumes a draft already exists. If it does not, use /draft first.

Run three passes. Each one looks for different things. Do not combine them, because fixing a problem in one pass often introduces a problem that belongs to another.

**First pass: accuracy and consistency.**

If research notes exist, check the draft against them. Did the drafting phase accurately represent what the sources say, or did it drift? A paraphrase that subtly changes a source's conclusion is worse than a missing citation, because the reader will trust it. Check numbers: if paragraph 3 says "a 40% increase" and paragraph 9 says "nearly half," make sure those refer to the same data and are both correct.

Check internal consistency. If a term is introduced in one section, make sure it means the same thing in later sections. If a position is attributed to a specific person or source, make sure the attribution stays accurate throughout. If the piece makes a claim early and then qualifies it later, make sure the qualification doesn't contradict the original claim.

**Second pass: banned patterns.**

Parallel structure. If three sentences in a row share the same grammatical shape (subject does X, subject does Y, subject does Z), rewrite one as a question, a fragment, or a sentence that starts with a verb. Three in a row is the ceiling.

Appositional comma splices. "A, who is B, does C while also D" loads a character's full biography before the first verb of action. Break it across two sentences.

Importance-announcing phrases. Cut "the key insight," "the critical distinction," "what matters here," "this is significant," and every synonym. If the prose needs to announce that a detail matters, the prose has failed to make it matter.

The "not X, but Y" frame. State what a thing is. Do not define it by negating a simpler version first.

Pedantic intensifiers. "Precisely because," "exactly why," "simply put." Cut all of them.

Hedging pileups. One "suggests" per paragraph maximum. "Might perhaps indicate" loses the reader. Pick one verb and commit.

One em dash per paragraph. One colon-then-elaborate per section.

Banned words. "Landscape," "nuanced," "robust," "leverage," "framework," "navigate," "unpack," "ecosystem," "holistic," "paradigm," "underscores," "highlights," "foundational," "multifaceted," "failure mode." Vague verbs of intellectual performance: "grapple with," "wrestle with," "reckon with," "sit with," "hold space for." If a specific word exists, use it. If no word is needed, cut the sentence.

**Third pass: rhythm, flow, and length.**

Check the opening. Does the first sentence put the reader inside the subject, or does it describe what the piece will be about? If it reads like an introduction to an introduction, cut it and start with the second or third sentence.

Read the text to catch monotone. Five declarative statements in a row at the same pitch read like a printout. After two or three assertions, shift gears: ask a question, quote someone, describe a scene, admit uncertainty, or let a sentence trail into implication.

Vary sentence length. A long subordinate-clause thought followed by a short sentence changes the texture. Two short sentences followed by one that takes a turn midway through keeps the reader awake.

Check paragraph shape. Claim, evidence, implication on repeat turns into wallpaper. Merge some paragraphs, break others, let a section run long and another run to two sentences.

Check transitions between sections. Does the reader know why the next section follows the previous one? Sometimes the connection is obvious from the content and needs no bridge. Sometimes a single question or linking sentence is needed. If a full transition paragraph exists and its only job is to announce the next topic, cut it.

Compare the last third of the piece against the first third. Long-form output degrades as the model gets further from the prompt. If the final sections are flatter, more formulaic, or more heavily stacked with declarative sentences, rewrite them to match the energy of the opening.

Try to cut 20%. Read each paragraph and ask whether the piece would lose anything if it disappeared. If not, remove it. If a paragraph survives but is four sentences long and only one sentence carries new information, keep that sentence and fold it into an adjacent paragraph. Patio11 writes long essays that feel dense because every paragraph advances the argument. Joan Didion writes short paragraphs that feel complete because nothing is wasted. Both approaches work. Padding does not.

End abruptly.
